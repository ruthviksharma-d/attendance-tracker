# Attendance Tracker

A responsive web-based attendance tracking application that allows users to easily record and review attendance records. The application provides a simple and intuitive interface for marking attendance, selecting dates, and viewing previously recorded attendance in a scrolling summary.

This project is built using **HTML, CSS, and JavaScript** and is designed to be lightweight and easy to use directly in a web browser.

---

## Features

- **Quick Entry:** Mark attendance quickly through a simple interface.
- **Date Selection:** Select specific dates for attendance entries.
- **Scrollable History:** View a scrollable list of attendance records.
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **No Dependencies:** Lightweight application with no external frameworks required.

---

## Tech Stack

- **HTML5** – Structure of the application
- **CSS3** – Styling and responsive layout
- **JavaScript (Vanilla JS)** – Application logic and interactivity
